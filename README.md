# 💊 Medical Store Management System

A browser-based pharmacy management system built with pure HTML, CSS, and JavaScript. No backend or installation required — just open in a browser!

## 🌟 Features

- 📁 **Excel/CSV Upload** — Import medicine inventory directly from Excel or CSV files
- 🔍 **Real-time Search** — Instantly search medicines as you type
- 🛒 **Cart System** — Add medicines, update quantities, remove items
- 🧾 **Receipt Generation** — Auto-calculated totals with print support
- 📱 **WhatsApp Sharing** — Share receipt directly via WhatsApp
- 🎨 **Modern UI** — Glassmorphism design with smooth animations

## 🛠️ Technologies Used

- HTML5
- CSS3 (Glassmorphism Design)
- JavaScript (Vanilla)
- [SheetJS](https://sheetjs.com/) — Excel/CSV file parsing

## 🚀 How to Use

1. Clone or download this repository
2. Open `index.html` in any browser
3. Upload your medicine Excel/CSV file (columns: Medicine Name, Price, Stock)
4. Search for medicines and add them to cart
5. Print or share receipt via WhatsApp

## 📂 Excel File Format

Your Excel file should have these columns:

| Medicine Name | Price | Stock |
|---------------|-------|-------|
| Paracetamol 500mg | 5.00 | 100 |
| Amoxicillin 250mg | 12.50 | 50 |
